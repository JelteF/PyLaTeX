pylatex.base_classes.latex_object¶

This module implements the base LaTeX object.

class pylatex.base_classes.latex_object.LatexObject(packages=None)¶

Bases: object

The class that every other LaTeX class is a subclass of.

This class implements the main methods that every LaTeX object needs. For conversion to LaTeX formatted strings it implements the dumps, dump and generate_tex methods. It also provides the methods that can be used to represent the packages needed.

Parameters:packages (list of Package instances) –

:
dumps()¶

Represent the class as a string in LaTeX syntax.

This method should be implemented by any class that subclasses this class.

Write the LaTeX representation of the class to a file.
